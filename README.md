# ASCII Art Convertor
___

### Basel Al-Dwairi

___

### Info:

A simple CLI script to turn a downloaded image or an online image into ASCII art 
that you can paste into chats.


Choose the mod in the script and run the code, pasting the path or the url of the image.
___

### Future plans :

- Add UI with streamlit
- Segmentation
- Normalisation
- Customizability
